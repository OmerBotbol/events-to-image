## Get Started

1. Clone this repository.
2. install pandas and scipy libs
3. Add the "cnvt_raw_2_csv" folder into the "utils" folder.
4. copy the folder that contain the .raw files into the "events" folder.
5. update the code in main.py according the instructions in it.
6. create "chunks" folders inside the folder that contain the .raw files
7. run main.py
